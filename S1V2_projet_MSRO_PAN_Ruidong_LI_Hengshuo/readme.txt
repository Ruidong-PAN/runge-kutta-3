Je vous remercie de lire cet page, ici je vous introduis comment je donne le chaque nom pour le fichies 

Pour le dossier code qui contient les source de code de cet projet. "shiyan.c"  "shiyan" ces sont les copie de code pour empêcher de supprimer ou modifier par erreur.

Pour le dossier image qui contient les image de loi normale et les images traitée par des données nous veut vérifier.
1: loi de normale (image dans eps)
      N(moyenne,varience) pour la fonction distribuée de loi normale.
      iN(moyenne,varience) pour les données on veut simuler le loi normale.
      2N(moyenne,varience) pour la fonction distrbuée de loi normale en dimension 2
      i2N(moyenne,varience) pour les données on veut simuler le loi normale en dimension 2

2: loi de uniforme(image dans eps)

      U(extrémité gauche,extrémité droite) pour la fonction de distrbuée de loi uniforme

      iU(extrémité gauche,extrémité droite) pour les donnéés on veut simuler la loi uniforme


Pour le dossier "gnu" dont je mets le fichier qui fonctionne avec "gnuplot" pour créer les images dedans 

1:  dat*.g on utilise quel fichier de données comme paramètre  pour créer le image.

2: ln*.g la loi de normale avec différent paramètre
   lu.g la loi de uniforme
   ln2.g la loi de normale en dimension deux 

Pour le quatrième dossier on met les données, 

1: dat* pour initiale paramètre
       m(moyenne): flottant 
       n(variance) : flottant
       nb(nombre de point on compte): entier 
       nbv(nombre de point sorte pour simuler le fonction de distribution): entier 
       la choix de choisie quel type de données tu a besoin (1: loi de normale, 2: loi de uniform , 3: loi de normale en dimension deux)

2: res*.txt ce qui corresponde de numéro de fichier "dat*"
